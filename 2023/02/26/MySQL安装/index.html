MySQL安装

1.安装

1.下载压缩包

MySQl版本5.7.19地址：https://cdn.mysql.com//archives/mysql-5.7/mysql-5.7.19-winx64.zip

解压目录：F:\Environment\mysql-5.7.19

2.配置环境变量

3.MySQL目录下新建mysql配置文件my.ini

  • 新建一个data文件夹

4.运行管理员模式下的CMD, 切换到mysql目录下的bin目录中,然后安装mysql服务

5.初始化数据库文件

6.启动mysql :

7.输入 mysql -u root -p 没有密码回车登录mysql

8.更改root密码, 然后刷新权限 输入flush privileges:

9.退出登录exit; 修改my.ini文件,删除最后一句skip-grant-tables(或者#注释掉), 这样以后登录需要输入密码了

10.重启mysql:

注意：二次安装

只需要将之前解压的，使用的MySQL5.7.19放到自己的环境目录，然后执行第二步和第四步，修改my.ini的路径
